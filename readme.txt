Sidebar widget that shows links to posts from exactly one year ago on the same date.

== Description ==

Plugin that shows links to posts that were posted exactly one year ago as a widget in the sidebar. Posts are listed by popularity based on the number of comments. 

If there were no posts posted a year ago then the widget will not show anything in the sidebar. 

User options: Title and number of posts to show (default "a year ago today" and "1").

== Installation ==

1. Upload `year_ago.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the widget menu to place the plugin in your sidebar.
